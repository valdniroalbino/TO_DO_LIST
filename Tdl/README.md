#Tarefas.java
Representa uma tarefa com os seguintes atributos:
- id → identificador único
- titulo → nome da tarefa
- data → data limite
- prioridade → normal ou prioritária
- status → estado da tarefa (começa sempre como "pendente")

#Lista.java
Gere todas as tarefas com os seguintes métodos:
- inserir() → adiciona uma nova tarefa
- ver() → mostra todas as tarefas
- atualizar() → atualiza um campo de uma tarefa
- remover() → remove uma tarefa
